# Rocket Trajectory Simulator

An advanced Python-based aerospace simulation platform for modeling rocket flight dynamics, multi-stage launch systems, planetary launches, and escape velocity analysis.

![Rocket Simulator Preview](simulation-preview.png)

---

## Features

### Flight Dynamics Simulation
- Real-time altitude vs time analysis
- Velocity vs time visualization
- Thrust-to-weight ratio calculations
- Flight duration analysis
- Stage separation tracking

### Aerospace Physics Modeling
- Gravity simulation
- Atmospheric drag analysis
- Fuel burn calculations
- Escape velocity comparison
- Multi-stage rocket simulation

### Planetary Launch Modes
Simulate launches from:
- Earth
- Moon
- Mars
- Venus
- Jupiter
- Titan

### Interactive Rocket Configuration
Adjust:
- Payload mass
- Fuel mass
- Dry mass
- Thrust output
- Burn time
- Multi-stage configurations

## Physics Concepts Applied

This simulator incorporates:

- Newton’s Second Law

F = ma

- Drag Force Equation

Fd = 1/2 * Cd * rho * A * v²

- Escape Velocity

ve = sqrt(2GM/r)

- Rocket Propulsion Principles
- Gravity and atmospheric effects
- Multi-stage launch mechanics
